# Quant Research & Portfolio Analytics Platform

A modular quantitative finance research platform built in Python for factor analysis, signal generation, portfolio construction, backtesting, and risk evaluation.  
Includes an interactive Streamlit dashboard for strategy exploration and performance visualization.

---

## Features

- Data pipeline for equities and benchmark data
- Factor models including momentum and volatility
- Signal generation and ranking system
- Portfolio construction strategies:
  - Equal weight
  - Top decile selection
  - Long-short portfolio
- Backtesting engine for strategy evaluation
- Risk analytics including:
  - Sharpe Ratio
  - Maximum Drawdown
  - Value at Risk (VaR)
  - Conditional VaR (CVaR)
  - Hit Ratio
- Interactive Streamlit dashboard (`app.py`)
